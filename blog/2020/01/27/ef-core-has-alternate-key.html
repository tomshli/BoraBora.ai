Entity Framework Core HasAlternateKey

January 27, 2020 9:00:00 PM

A database table can have multiple unique keys including the primary key. A foreign key can reference one of the unique keys. In Entity Framework Core, use the HasAlternateKey fluent API to specify the unique key.

This fluent API is only available on EF Core and in unavailable in EF6.
